lightmvc1.0 Java (Hibernate+SpringMVC+easyui) 示例
技术群：189811613

开发环境:

1) 数据库：mysql 
请修改 resources/config.properties这个文件的数据库连接
创建数据库lightmvc
导入测试数据：sql/mysql/init.sql
mysql -u root -p lightmvc < init.sql
2) JDK 1.6
3) 项目文件UTF-8编码，注意把工作空间编码设成UTF-8
4) 启动tomcat
